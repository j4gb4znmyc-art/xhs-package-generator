# 得物多风格图文提示词生成器

这是一个本地可运行、也可部署到 Streamlit Cloud 的「得物图文多风格提示词生成器」。

工具不直接生成图片，不接入任何图片 API，也不需要 OpenAI API Key。它根据用户上传的产品图和产品信息，输出一份可以交给 ChatGPT 继续执行的结构化提示词包：每个风格组包含「5 张独立图文策划 + 1 篇种草文章」。

## 功能

- 上传产品图并本地预览
- 支持 `jpg/jpeg/png/webp/heic/heif`
- HEIC/HEIF 可自动转换为预览图
- 输入产品名称、品牌名、核心卖点
- 可补充品类、使用场景、目标用户、功效依据、禁用词、额外要求
- 支持 1 到 10 组风格输出
- 支持全部风格、单风格、多风格选择
- 每组都生成可单独复制给 ChatGPT 的执行提示词
- 输出 Markdown 指令包
- 输出 JSON 备份
- 本地保存最近一次生成结果

## 输出结构

```text
# 得物多风格图文提示词生成结果
## A. 产品信息摘要
## B. 本次选用风格列表
## C. 分组输出
```

每个分组包含：

- 风格 ID 和风格名称
- 风格定位说明
- 本组差异锚点
- 本组建议内容模块
- 5 张图片策划
- 5 个备选标题
- 1 篇完整文章
- 推荐标签
- 可复制的本组执行提示词

## 本地运行

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
streamlit run app.py
```

打开：

```text
http://localhost:8501
```

## Streamlit Cloud 部署

仓库根目录需要包含：

```text
app.py
requirements.txt
README.md
```

部署参数：

```text
Branch: main
Main file path: app.py
```

继续推送到同一个 GitHub 仓库后，Streamlit Cloud 会自动更新，公网网址保持不变。

## HEIC/HEIF

项目通过 `pillow-heif` 转换 HEIC/HEIF。如果部署环境安装失败，工具仍可运行，但会提示用户手动转换 JPG 后再上传。
